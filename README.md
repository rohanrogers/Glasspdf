# GlassPDF

A privacy-focused suite of PDF, Image, and Document utilities built on Next.js 15. The project features local-first, browser-based processing, ensuring NO data leaves your machine.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:9002](http://localhost:9002) in your browser.
