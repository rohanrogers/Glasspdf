# GlassPDF

Local-first document tools. Private by design.

GlassPDF is a suite of professional document and image utilities that runs entirely within your web browser. Whether you are manipulating PDFs, previewing PSD files, or converting images, every operation is performed locally on your device. Your files are never uploaded, stored, or processed on an external server.

## Privacy

Your data belongs to you. GlassPDF guarantees zero data collection and zero server-side processing. Because all computations occur within the isolated environment of your browser, your files remain strictly on your machine. We cannot see, track, or access your documents, ensuring absolute confidentiality.

## Features

- **Merge PDF**: Combine multiple documents seamlessly.
- **Split PDF**: Extract pages into distinct files.
- **Compress PDF**: Reduce file size without sacrificing clarity.
- **Image Compressor**: Optimize images for web and print.
- **PSD Preview**: View Adobe Photoshop documents directly in the browser.
- **Page Rearrangement**: Reorder and organize PDF layouts effortlessly.

## Technology

GlassPDF leverages Next.js and modern browser APIs to deliver desktop-class performance on the web. By utilizing WebAssembly and advanced client-side rendering engines, heavy file processing that traditionally required backend servers is now executed instantly and securely on your own hardware.

## Open Source

GlassPDF is open-source software designed for the open web. It is built upon powerful open-source engines and community projects. All underlying components adhere to their respective open-source licensing.
